# Yogya

Yogya is an exam eligibility and tracking platform.

## Project Structure
- `yogya_app/`: The Flutter mobile application containing the full frontend implementation.
- `Yogya_Design_Proposal.pdf`: Detailed design proposal and wireframes.
- Additional python utility scripts.

## Development

The application is built using:
- Flutter / Dart
- Riverpod for State Management
- GoRouter for Navigation
- Clean Architecture principles
